Authentication System

This is a general authentication system that can be used in any web application. I have used 'bcrypt' for hashing passwords, and Passport.js for authentication. The user can either Sign in using his existing Google accounts, or can create a new account with an existing email id. Google Recaptcha V2 has been used for verification during Sign in. Once logged in, the user can change his password. There is an option for getting a new password through a recovery email in case the user forgets their password.

Frameworks/libraries : NodeJS, MongoDB, ExpressJS,Passport.js, Nodemailer, Noty
